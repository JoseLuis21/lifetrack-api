# Life Tracker API
Life Track is a platform where you can track habits and get useful insights from them.

## Technology Stack
- Go 1.15
- Hashicorp Terraform
- OAuth2 (Google, Facebook & Apple)
- AWS
  - Route53
  - API Gateway
  - Cognito
  - Lambda
  - DynamoDB
  - S3 Intelligent Tier
  - CLI v2
  - IAM
